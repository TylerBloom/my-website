About

Quartz is a library for scientific computing for Rust. It includes a port of most of the major features of the Python library SciPy. There are also tools for infacing with other MtG-related things, such as deck websites.

The name - "scientific" is often shorted to "sci" or "sy". If shortened to "si", you can substitute "Si" for "Fe" in Rust's chemical formula to "SiO_2", the primary component of Quartz.

Impetus

Rust's features of memory and concurrency safty, lack of GC, and speeds similar to those of C/C++ make it an excellent canidate for the scientific computing space. Moveover, there are not very few language offer well-rounded libraries to aid with scientific computing. Python with its SciPy library in an exception to this.

For people getting into computing for scientific projects, SciPy offers a fantastic user expirence in a language that is relately easy to learn. However, Python is not a great language for performance, which is vital for larger projects. Quartz's primary goal is to combine the wonderful features of Rust with the bevy of tools similar to what is available in SciPy to people in the scientific computing space.
